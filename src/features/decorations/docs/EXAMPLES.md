## Writing Tips from Examples

### Quantification Patterns
- Personnel: "led 28 controllers," "managed 180 operators"
- Financial: "$450 million program," "saved $1.2 million"
- Operations: "47,000 movements," "2,100 combat sorties"
- Metrics: "99.99 percent availability," "zero Class A mishaps"
- Scope: "across five flights," "supporting four combatant commands"

### Strong Action Verbs by Rank

**Junior Enlisted (AB-SrA):**
- Processed, maintained, supported, ensured, coordinated

**NCO (SSgt-TSgt):**
- Directed, managed, developed, led, orchestrated, oversaw

**SNCO (MSgt-CMSgt):**
- Established, spearheaded, transformed, championed, pioneered

**Officers:**
- Commanded, directed, forged, enabled, delivered

### Impact Language
- "directly contributing to mission readiness"
- "protecting [X] personnel and [$X] enterprise"  
- "ensuring zero [mishaps/incidents/delays]"
- "enabling rapid [deployment/response/execution]"
- "strengthening [alliance/partnership/capability]"
